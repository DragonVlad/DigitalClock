# Digital Clock

Этот проект представляет собой реализацию цифровых часов на веб-странице. Часы отображают реальное время. Он демонстрирует использование работу с датой/временем в Javascript,
а также асинхронность выполнения кода.

## Основные возможности:
- Отображение времени: точное отображение часов, минут и секунд.
